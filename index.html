Overwolf

Overwolf – software platform designed to help developers create extensions for video games, which are then offered to users through Overwolf's App Store. The platform was created by Overwolf Ltd., and extensions are often focused on providing in-game services that would normally require a user to exit the game, such as the use of a web browser or an IM client. Other extensions provide game-specific features that can remind users about certain in-game events, easing the game experience.
